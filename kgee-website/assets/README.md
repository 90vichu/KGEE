# KGEE 2026 Website - Asset Setup Guide

## Required Assets

This directory should contain:

### Logo Files
1. **kgee-logo.png**
   - File: Official KGEE 2026 logo (full color)
   - Used in: Header navigation, home page hero section
   - Recommended size: 400x200px or higher
   - Format: PNG with transparency
   - Source: Place the provided KGEE 2026 logo file here

2. **kgee-logo-white.png**
   - File: White version of KGEE 2026 logo
   - Used in: Footer
   - Recommended size: 300x150px or higher
   - Format: PNG with transparency
   - Source: Create from the original logo or provide white variant

## How to Add Assets

1. Save the official KGEE 2026 logo file as `kgee-logo.png` in this directory
2. Create or save a white version as `kgee-logo-white.png`
3. Ensure both files have proper transparency (PNG format)
4. Do not compress excessively - maintain quality

## Future Asset Expansion

This directory can be expanded with:
- `images/` - Subdirectory for gallery images
- `icons/` - SVG icons (currently using inline SVG)
- `fonts/` - Custom fonts (currently using Google Fonts)
- `videos/` - Promotional videos

## Notes

- All images should be web-optimized
- Use PNG for images with transparency
- Use JPG for photography/gallery images
- Use WebP for modern browsers (with fallbacks)
- Consider lazy-loading for images below the fold

For more information, see README.md in the parent directory.
